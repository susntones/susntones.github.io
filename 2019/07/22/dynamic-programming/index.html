Dynamic Programming

处理DP问题的两种方式

  • 记忆化搜索
  • 循环，包括自底向上和自顶向下两种方式。

DP常见的问题类型

  • matrix dp
  • sequence
  • two sequence dp
  • backpack

动态规划的4点要素

  • 状态 State

    存储小规模问题的结果。

  • 方程 Function

    状态之间的联系，怎样通过小的状态，来算大的状态。

  • 初始化 Intialization

    最极限的小状态

  • 答案 Answer

    最终状态
